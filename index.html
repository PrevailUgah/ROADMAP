Scenario: User Login Credentials

Exploiter Action (Fraud Mindset)

**Action:** Sets up a fake login page (Phishing) to capture usernames and weak passwords (e.g., from old forum leaks).

**Skill Used:** Basic HTML/CSS manipulation.

Architect Solution (Builder Mindset)

**Action:** Implements **OAuth 2.0** and **bcrypt hashing** to ensure passwords are never stored in plaintext and unauthorized access is impossible.

**Skill Used:** Phase 3 Security & Compliance.

Scenario: Detecting Stolen Cards

Exploiter Action (Fraud Mindset)

**Action:** Uses automation scripts to test thousands of stolen credit card numbers against weak merchant payment gateways until one succeeds.

**Skill Used:** Simple Python scripting (Automation).

Architect Solution (Builder Mindset)

**Action:** Develops a **Machine Learning model** (Phase 4) that flags transactions based on velocity, location, and amount patterns, blocking the fraud attempt instantly.

**Skill Used:** Phase 4 Data & AI.
